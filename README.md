# Welcome

To build:
```sh
gradle build assemble
```

To run:
```sh
java -jar build/libs/PersonsFinder-0.0.1-SNAPSHOT.jar
```

To run simple test suite:
```sh
sh tests.sh
```

To manually test endpoints, open the postman collection: `./sqaure1.postman_collection.json`
